# Project Overview
This project is designed to strengthen your understanding of React Hooks, state management, and API integration.
It covers the core concepts of modern React development while interacting with a local JSON database (db.json) to perform full CRUD operations.

## Learning Objectives
Gain practical experience with key React Hooks:
useState
useEffect
useCallback
useContext
Custom Hooks
Manage and share state effectively across multiple components.
Learn to handle asynchronous operations using fetch or axios.
Implement CRUD functionality:
GET – Fetch data from the API
POST – Add new data
PUT – Update existing data
DELETE – Remove data
Understand component communication and prop drilling.
Build clean, reusable logic using custom hooks.
